# NumberNinja

Number Ninja does not collect, store, or share any personal information.
All progress data, including quiz scores and homework tracking, is stored locally on the user’s device using Apple’s built-in storage systems. This information never leaves the device.
Number Ninja:
Does not require account creation
Does not use third-party analytics
Does not display advertisements
Does not collect location data
Does not share any data with third parties
